# LRR CLARIFICATION: The Arborec
## TYPE: Agent
### COMPONENT: Letani Ospha
#### ID: the_arborec_letani_ospha

1. The replaced ship is not destroyed. The replacing ship is not produced.
2. TheSarween Toolstechnology and similar effects will not allow a more expensive ship to be placed.
3. If the player wishes to place a ship, but there are none of that type left in their reinforcements, they may remove a ship of that type from any system that does not contain one of their command tokens and place that instead. This ship will be placed undamaged.
4. A fighter may be placed with Letani Ospha.
