# LRR CLARIFICATION: The Mahact Gene–Sorcerers
## TYPE: Commander
### COMPONENT: Il Na Viroset
#### ID: the_mahact_genesorcerers_il_na_viroset

1. The Mahact player cannot use any “when/after you activate a system” abilities.
2. Other players cannot use any “when/after another player activates a system that…” abilities.
3. If the Mahact player activates a system containing another player’s units, they must still return any promissory notes with such an activation as the return condition.
4. The Mahact player may still use any “at the end of your turn” abilities.
