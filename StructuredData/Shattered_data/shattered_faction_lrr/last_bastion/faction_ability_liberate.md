# LRR CLARIFICATION: Last Bastion
## TYPE: Faction Ability
### COMPONENT: Liberate
#### ID: last_bastion_liberate

1. If the Bastion player gains control of an uncontrolled planet, they may choose if they resolveLiberatebefore or after they explore that planet.
2. If they explore first, and the number of infantry or the resource value of the planet changes while resolving the exploration, then the new values will be used when resolvingLiberate.
3. The number of mechs is irrelevant for determining which effect to resolve.
