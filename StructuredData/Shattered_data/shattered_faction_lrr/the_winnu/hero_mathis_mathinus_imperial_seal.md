# LRR CLARIFICATION: The Winnu
## TYPE: Hero
### COMPONENT: Mathis Mathinus — Imperial Seal
#### ID: the_winnu_mathis_mathinus_imperial_seal

1. A player chosen to perform the secondary ability must still spend a command token from their strategy pool to do so, except for theLeadershipstrategy card and potentially theTradestrategy card.
2. In the case of theConstructionstrategy card, the command token is placed as required on the game board.
3. If the Winnu player chooses theTradestrategy card for this ability and the Winnu player chooses a player to resolve the secondary without spending a command token via the primary ability, the chosen player must resolve the secondary ability. Additionally, if the Winnu player chooses a player to resolve the secondary via the Hero’s ability, it is optional for that player to resolve the secondary, but if they do, they must spend a strategy token to do so.
