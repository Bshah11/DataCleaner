# LRR CLARIFICATION: The Barony of Letnev
## TYPE: Commander
### COMPONENT: Rear Admiral Farran
#### ID: the_barony_of_letnev_rear_admiral_farran

1. If the Letnev player has more ships in a system than allowed for by their fleet pool plusArmada, they must first remove ships to meet this limit before they can unlock Rear Admiral Farran.
2. This effect may be applied when a Dunlain Reaper uses itsSustain damageability.
3. If the Ul player has the Letnev player’sAlliancepromissory note, then this effect may be applied when a Hel–Titan PDS uses itsSustain damageability.
4. A unit that becomes damaged without using itsSustain Damageability will not trigger this effect.
5. If multiple units use theirSustain Damageability within a single step, the trade goods are gained one at a time.
