# COMPONENT CLARIFICATION: Agendas
## TYPE: Agenda
### COMPONENT: Shared Research
#### ID: agendas_shared_research | Source: Base Game

1. If theShared Researchlaw is in play, a player still cannot move into a nebula unless it is the active system.
2. If theShared Researchlaw is in play, a ship that begins theMovementstep of a tactical action in a nebula still treats its move value as one for the duration of that step.
