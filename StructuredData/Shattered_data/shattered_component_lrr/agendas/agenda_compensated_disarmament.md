# COMPONENT CLARIFICATION: Agendas
## TYPE: Agenda
### COMPONENT: Compensated Disarmament
#### ID: agendas_compensated_disarmament | Source: Base Game

1. All trade goods are gained together.
2. Each destroyed infantry will provide one trade good and each destroyed mech will provide one trade good.
3. A planet with zero units on it is eligible to be elected.
4. Structures are unaffected byCompensated Disarmament.
