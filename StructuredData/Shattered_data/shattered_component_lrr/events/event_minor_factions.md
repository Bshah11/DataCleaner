# COMPONENT CLARIFICATION: Events
## TYPE: Event
### COMPONENT: Minor Factions
#### ID: events_minor_factions | Source: Base Game

1. During setup, if a player places a minor faction system with two planets, that player chooses which planet gets two neutral infantry and which planet gets one.
2. During setup, if a player is dealt the Creuss reference card, they place the Creuss Gate (tile 17) in the equidistant, and place the Creuss home system (tile 51) in the common play area.
3. During setup, if a player is dealt the Rebellion reference card, they place the Sorrow (tile 94) in the equidistant, and place the Rebellion home system (tile 118) in the common play area.
4. During setup, if a player is dealt the Keleres reference card, then they select a home system at random from amongst the Argent, Mentak and Xxcha, excluding any that have already been placed.
5. If Mahact exists as a minor faction, the player that controls Ixth gains access to the ability of Il Na Viroset, the Mahact commander.
6. Commander abilities of minor factions do not need to be unlocked; they are available as soon as a player takes the corresponding alliance card.
7. When a player gains control of a minor faction planet, if it is not controlled by another player, they explore it. They choose which planet trait they explore it as.
8. A player only loses a minor faction’s alliance card if a single other player controls each planet in that minor faction’s system.
9. The planets in a minor faction’s system are not home planets, nor is the system a home system.
10. All neutral infantry have a combat value of 8, and otherwise have no special abilities, regardless of which minor faction they belong to.
11. Minor factions are not players.
