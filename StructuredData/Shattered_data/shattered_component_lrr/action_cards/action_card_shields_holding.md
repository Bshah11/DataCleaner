# COMPONENT CLARIFICATION: Action Cards
## TYPE: Action_card
### COMPONENT: Shields Holding
#### ID: action_cards_shields_holding | Source: Base Game

1. Shields Holdingcannot be used to cancel hits produced bySpace Cannonabilities.
2. Shields Holdingcannot be used to prevent an effect that directly destroys a unit.
3. Shields Holdingcannot be used during ground combat.
4. Shields Holdingmay be used to cancel hits produced by any ability during space combat.
5. A player cannot play a secondShields Holdingto cancel a second pair of hits produced by the same dice roll or other source.
