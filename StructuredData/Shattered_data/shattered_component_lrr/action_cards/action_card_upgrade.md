# COMPONENT CLARIFICATION: Action Cards
## TYPE: Action_card
### COMPONENT: Upgrade
#### ID: action_cards_upgrade | Source: Base Game

1. The cruiser must be in the system when it is activated forUpgradeto be played.Upgradecannot be played if the cruiser is moved into the active system later in the turn.
2. A player must have at least one cruiser in the active system to play and resolveUpgrade.
3. If a player wishes to place a dreadnought, but there are none left in their reinforcements, they may remove a dreadnought from any system that does not contain one of their command tokens and place that instead. This dreadnought will be placed undamaged.
