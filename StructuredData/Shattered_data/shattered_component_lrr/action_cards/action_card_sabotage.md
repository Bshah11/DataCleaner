# COMPONENT CLARIFICATION: Action Cards
## TYPE: Action_card
### COMPONENT: Sabotage
#### ID: action_cards_sabotage | Source: Base Game

1. If a player has a second copy of an action card, they may play it if the first was cancelled bySabotage.
2. If an action card is cancelled bySabotage, no costs are paid.
3. A player playing an action card announces the following beforeSabotagemay be played:
4. Any dice are rolled and any cards are drawn only after all other players has declined to playSabotage.
5. If an action card instructs a player to research a technology, that player need not announce what technology they are researching until all other players has declined to playSabotage.
6. If an action card is played that will have an effect later, such as a rider, that card may only be cancelled bySabotagewhen it is originally played.
7. If the action card that was cancelled bySabotagewas to be used to perform a component action, the active player must perform a different action, or pass.
