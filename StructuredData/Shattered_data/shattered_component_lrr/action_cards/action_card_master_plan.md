# COMPONENT CLARIFICATION: Action Cards
## TYPE: Action_card
### COMPONENT: Master Plan
#### ID: action_cards_master_plan | Source: Base Game

1. Effects that trigger at the start or end of a player’s turn will not trigger between each of those player’s actions.
2. If a player’s turn is ended by a game effect, they cannot useMaster Planto perform an additional action.
3. A player cannot useMaster Planto pass on the same turn they performed an action.
4. If a player’s second action would be to use a component action to play an action card, and that card is cancelled, they are not required to perform an alternate second action on that turn.
