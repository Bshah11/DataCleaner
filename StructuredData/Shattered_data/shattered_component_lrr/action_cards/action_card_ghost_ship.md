# COMPONENT CLARIFICATION: Action Cards
## TYPE: Action_card
### COMPONENT: Ghost Ship
#### ID: action_cards_ghost_ship | Source: Base Game

1. A player may place the destroyer in the Creuss gate (tile 17), the wormhole nexus, or any system with a printed wormhole or wormhole token.
2. The destroyer cannot be placed in the home system of an eliminated player.
3. There must be an eligible system to place the destroyer in for a player to play and resolveGhost Ship.
4. If a player wishes to place a destroyer, but there are none left in their reinforcements, they may remove a destroyer from any system that does not contain one of their command tokens and place that instead.
