# COMPONENT CLARIFICATION: Objectives
## TYPE: Objective
### COMPONENT: Make History
#### ID: objectives_make_history | Source: Base Game

1. The number of units in the system is irrelevant, if there is at least one.
2. A ship, ground force, or structure will all qualify forMake History.
3. If Mirage is placed in an anomaly, or theNano–Forgerelic is attached to a planet in an anomaly, that system will only count as one system towards the requirements ofMake History.
4. Any system containing a Vuil’raith player’s Dimensional Tear is a gravity rift, and thus, an anomaly.
