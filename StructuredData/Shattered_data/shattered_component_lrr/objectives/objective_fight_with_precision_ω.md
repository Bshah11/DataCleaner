# COMPONENT CLARIFICATION: Objectives
## TYPE: Objective
### COMPONENT: Fight with Precision Ω
#### ID: objectives_fight_with_precision | Source: Codex

1. ScoringFight with Precision Ωwill count as that player’s single secret objective scoring for that combat.
2. Fight with Precisionis scored immediately in theAnti–Fighter Barragestep of combat.
3. If a player’s last fighter in a system is destroyed, but an ability allows that player to then place fighters into that system,Fight with Precision Ωmay still be scored.
4. Any effect that destroys fighters during theAnti–Fighter Barragestep may be used to scoreFight with Precision Ω, not just hits produced by an anti–fighter barrage roll.
