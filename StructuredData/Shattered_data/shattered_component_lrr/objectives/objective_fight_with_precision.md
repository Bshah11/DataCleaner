# COMPONENT CLARIFICATION: Objectives
## TYPE: Objective
### COMPONENT: Fight with Precision
#### ID: objectives_fight_with_precision | Source: Base Game

1. ScoringFight with Precisionwill count as that player’s single secret objective scoring for that combat.
2. Fight with Precisionis scored immediately in theAnti–Fighter Barragestep of combat.
3. If a player’s last fighter in a system is destroyed, but an ability allows that player to then place fighters into that system,Fight with Precisionmay still be scored.
