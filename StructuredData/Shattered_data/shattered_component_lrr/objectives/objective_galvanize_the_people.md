# COMPONENT CLARIFICATION: Objectives
## TYPE: Objective
### COMPONENT: Galvanize the People
#### ID: objectives_galvanize_the_people | Source: Base Game

1. The command tokens must be spent during the status phase. Any command tokens that were spent during the action phase will have no effect.
2. Any effect that triggers when a command token is spent will be triggered six times. For example, if the Muaat player has unlocked their commander, Magmus, they will gain six trade goods, one at a time, when scoringGalvanize the People.
