# COMPONENT CLARIFICATION: Technology
## TYPE: Technology
### COMPONENT: Fleet Logistics
#### ID: technology_fleet_logistics | Source: Base Game

1. Effects that trigger at the start or end of a player’s turn will not trigger between each of those player’s actions.
2. A player may use the ability ofFleet Logisticsthe same turn they gain it.
3. If a player’s turn is ended by a game effect, they cannot useFleet Logisticsto perform an additional action.
4. A player cannot useFleet Logisticsto pass on the same turn they performed an action.
5. If a player’s second action would be to use a component action to play an action card, and that card is cancelled, they are not required to perform an alternate second action on that turn.
