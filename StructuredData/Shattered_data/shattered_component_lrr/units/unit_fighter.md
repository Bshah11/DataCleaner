# COMPONENT CLARIFICATION: Units
## TYPE: Unit
### COMPONENT: Fighter
#### ID: units_fighter | Source: Base Game

1. Fighters and ground forces do not count towards capacity during combat. As such, any number of Fighter II may be in a system during combat, regardless of capacity and fleet pool. This will usually only be caused by effects that place units in a system during combat. When combat ends however, units will need to be removed to meet capacity and fleet pool requirements.
2. A ship cannot move through a system that contains Fighter II that are controlled by another player during theMovementstep of a tactical action.
3. Ships may be moved into, produced in or placed in a system even if doing so would exceed the fleet pool limit. However, some of those ships must immediately be removed to satisfy the limit.
