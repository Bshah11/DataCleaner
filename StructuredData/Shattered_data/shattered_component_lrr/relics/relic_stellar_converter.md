# COMPONENT CLARIFICATION: Relics
## TYPE: Relic
### COMPONENT: Stellar Converter
#### ID: relics_stellar_converter | Source: Base Game

1. The chosen planet must be in a system adjacent to the unit withbombardment; the planet and unit cannot both be in the same system.
2. A system that contains a planet destroyed byStellar Converter, and no other planets, is considered to contain no planets.
3. A player cannot target a planet they control withStellar Converter.
