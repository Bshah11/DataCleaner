# COMPONENT CLARIFICATION: Exploration Cards
## TYPE: Exploration_card
### COMPONENT: Hazardous Relic Fragment
#### ID: exploration_cards_hazardous_relic_fragment | Source: Base Game

1. If the relic deck is empty, a player may still purge three Hazardous Relic Fragments as an action.
2. If an eliminated player had any Hazardous Relic Fragments, they are discarded to the hazardous planet exploration discard pile.
