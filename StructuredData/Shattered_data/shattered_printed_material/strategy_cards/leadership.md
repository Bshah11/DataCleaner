# Strategy Cards: Leadership

* **Card Id**: STRAT_LEADERSHIP
* **Initiative**: 1
## Primary
* **Gain Tokens**: Gain 3 command tokens.
* **Influence Conversion**: Spend any amount of influence to gain 1 command token for every 3 influence spent.
## Secondary
* **Influence Conversion**: Spend any amount of influence to gain 1 command token for every 3 influence spent.
