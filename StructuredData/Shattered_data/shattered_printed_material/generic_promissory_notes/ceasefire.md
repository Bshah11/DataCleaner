# Generic Promissory Notes: Ceasefire

* **Note Id**: PROMISSORY_CEASEFIRE
* **Source**: base
* **Type**: standard
* **Timing**: After the {{color}} player activates a system that contains 1 or more of your units
* **Effect**: The {{color}} player cannot move units to the active system. Then return this card to the {{color}} player.
## Metadata
* **Sabotage Window**: After the system is activated, but before the Movement step begins.
* **Clarifications**: ['The activation token remains in the system.', 'Units already in the system cannot move, and units from other systems cannot enter.']
