# Action Cards: Unstable Planet

* **Card Id**: AC_UNSTABLE_PLANET
* **Quantity**: 1
* **Phase**: ACTION
* **Source**: Base
* **Timing**: ACTION: Choose 1 hazardous planet.
* **Effect**: Exhaust that planet and destroy up to 3 infantry on it.
## Metadata
* **Sabotage Window**: After planet is chosen.
* **Clarifications**: ['Can be played if: Planet is exhausted.', 'ERRATA: UP TO 3 infantry. Can destroy 0 infantry.', 'Cannot target own planets.']
