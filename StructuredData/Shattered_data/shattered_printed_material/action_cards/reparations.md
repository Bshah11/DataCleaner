# Action Cards: Reparations

* **Card Id**: AC_REPARATIONS
* **Quantity**: 1
* **Phase**: ACTION (INVASION)
* **Source**: Base
* **Timing**: After another player gains control of a planet you control
* **Effect**: Exhaust 1 planet that player controls and ready 1 planet you control.
## Metadata
* **Sabotage Window**: After card is played.
* **Clarifications**: ['Must target ready enemy planet, if able.', 'Must target exhausted own planet, if able.']
