# Action Cards: Signal Jamming

* **Card Id**: AC_SIGNAL_JAMMING
* **Quantity**: 1
* **Phase**: ACTION
* **Source**: Base
* **Timing**: ACTION: Choose 1 non-home system that contains or is adjacent to 1 of your ships.
* **Effect**: Place a command token from another player's reinforcements in that system.
## Metadata
* **Sabotage Window**: After the system is chosen.
* **Clarifications**: ['Cannot place own token.']
