# Action Cards: Morale Boost

* **Card Id**: AC_MORALE_BOOST
* **Quantity**: 4
* **Phase**: ACTION (SPACE COMBAT/INVASION)
* **Source**: Base
* **Timing**: At the start of a combat round
* **Effect**: Apply +1 to the result of each of your unit's combat rolls during this combat round.
## Metadata
* **Sabotage Window**: Before making rolls.
* **Clarifications**: ['Affects rerolls as well.']
