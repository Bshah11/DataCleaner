# Action Cards: Lost Star Chart

* **Card Id**: AC_LOST_STAR_CHART
* **Quantity**: 1
* **Phase**: ACTION (ACTIVATION)
* **Source**: Base
* **Timing**: After you activate a system
* **Effect**: During this tactical action, systems that contain alpha and beta wormholes are adjacent to each other.
## Metadata
* **Sabotage Window**: Before moving ships.
* **Clarifications**: ['Works even if the agenda that disables wormholes is in play.']
