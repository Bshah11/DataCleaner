# Action Cards: Skilled Retreat

* **Card Id**: AC_SKILLED_RETREAT
* **Quantity**: 4
* **Phase**: ACTION (SPACE COMBAT)
* **Source**: Base
* **Timing**: At the start of a combat round
* **Effect**: Move all of your ships from the active system into an adjacent system that does not contain another player's ships; the space combat ends in a draw. Then, place a command token from your reinforcements in that system.
## Metadata
* **Sabotage Window**: Before moving ships.
* **Clarifications**: ['Draw is neither win or loss for either party.', 'Move all ships. Even fighter I.', 'Graphic: https://i.imgur.com/DFy5dk1.png']
