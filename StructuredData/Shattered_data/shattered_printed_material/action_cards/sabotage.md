# Action Cards: Sabotage

* **Card Id**: AC_SABOTAGE
* **Quantity**: 4
* **Phase**: ANY
* **Source**: Base
* **Timing**: When another player plays an action card other than 'Sabotage'
* **Effect**: Cancel that action card.
## Metadata
* **Sabotage Window**: Cannot be sabotaged.
* **Clarifications**: ['After choices, before dice rolls.']
