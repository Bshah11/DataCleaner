# Action Cards: Tactical BOMBARDMENT

* **Card Id**: AC_TACTICAL_BOMBARDMENT
* **Quantity**: 1
* **Phase**: ACTION
* **Source**: Base
* **Timing**: ACTION: Choose 1 system that contains 1 or more of your units that have BOMBARDMENT.
* **Effect**: Exhaust each planet controlled by other players in that system.
## Metadata
* **Sabotage Window**: After system is chosen.
* **Clarifications**: ['Can target system that contain no planets other players control.']
