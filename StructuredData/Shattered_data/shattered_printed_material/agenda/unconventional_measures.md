# Agenda: Unconventional Measures

* **Card Id**: AGENDA_UNCONVENTIONAL_MEASURES
* **Type**: Directive
* **Source**: base
* **Effect Pass**: Each player that voted 'For' draws 2 action cards.
* **Effect Fail**: Each player that voted 'For' discards all of his action cards.
## Metadata
* **Clarifications**: ['Regular hand size limit applies.']
