# Agenda: Clandestine Operations

* **Card Id**: AGENDA_CLANDESTINE_OPERATIONS
* **Type**: Directive
* **Source**: pok
* **Effect Pass**: Each player removes 2 command tokens from their command sheet and return those tokens to their reinforcements.
* **Effect Fail**: Each player removes 1 command token from their fleet pool and returns that token to their reinforcements.
