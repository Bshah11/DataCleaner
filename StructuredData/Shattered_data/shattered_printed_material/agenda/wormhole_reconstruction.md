# Agenda: Wormhole Reconstruction

* **Card Id**: AGENDA_WORMHOLE_RECONSTRUCTION
* **Type**: Law
* **Source**: base
* **Effect Pass**: All systems that contain either an alpha or beta wormhole are adjacent to each other.
* **Effect Fail**: Each player places a command token from his reinforcements in each system that contains a wormhole and 1 or more of his ships.
## Metadata
* **Repeal Interaction**: Alpha and Beta wormholes are no longer mutually adjacent.
* **Clarifications**: ['Standard adjacency rules (Alpha-to-Alpha and Beta-to-Beta) resume.']
