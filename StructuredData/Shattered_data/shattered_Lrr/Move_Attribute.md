# Move (Attribute)
## Specific rule - [Move is an attribute of some units that is presented on faction sheets and unit upgrade technology cards.]
### Rule points
1. A unit’s move value indicates the distance from its current system that it can move during the Movement step of a tactical action.

### Notes
- When an effect moves a unit to an adjacent system, modifying that unit’s move value will have no effect.
